How to use flags

Chem:nomenclature:

+3d = switches to 3d mode, can be used with other flags for the 2d image, basically this shows the 3d model first if you have this flag instead of the image, however image flags like c, h, m can be used
st = stereochemistry image
d = default, used like this chem:histamine/d-c+m+h+q-p: basically adds or removes specific settings from the default image settings
p = flip vertical
q = flip horizontal
c = show carbons
h = show hydrogens
when used like chem:histamine/+c+h:, it overrides default settings and uses the settings only after the / which would be show carbons and hydrogens
