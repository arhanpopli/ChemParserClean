Image Size Controls Test

Test page for the Chemistry Extension's new image size control feature

Testing Instructions

  1. Load this extension in Chrome (chrome://extensions/)
  2. Open the extension popup and enable "Save Size Per Page" or "Save Size By SMILES"
  3. Hover over any molecule image below to see the size control arrows in the bottom-left corner
  4. Click the up arrow (â–²) to increase size, down arrow (â–¼) to decrease size
  5. Reload the page to verify that your size preferences are saved
  6. Try different combinations of the two save options to test the behavior

Features to Test

  • Size control arrows appear on hover (bottom-left corner)
  • Up arrow increases image size by 20px
  • Down arrow decreases image size by 20px
  • Size is constrained between 100px and 800px
  • With "Save Size Per Page" enabled: each page remembers its own sizes
  • With "Save Size By SMILES" enabled: all instances of the same molecule use the same size across all pages
  • Sizes persist across page reloads
