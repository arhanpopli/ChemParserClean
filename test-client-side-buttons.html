What Changed:

Previously, client-side rendering would try to add control buttons directly to the <img> element when there was no rotation. Since <img> elements cannot have child elements, the buttons wouldn't appear.

Now, the code always creates a wrapper <div> container, which can properly hold the control buttons as children, just like MoleculeViewer and mol2chemfig renderers do.
